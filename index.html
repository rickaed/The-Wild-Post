le HTML (Hyper Text Markup Language => language de balises hypertext pour les anglophobe).
concerne le contenu texte et image, ainsi que la structure global de votre page web préféré.
le squelette en somme.

Le CSS (Cascading Style Sheets => feuille de style en cascade pour les anglophobes).
permet de créer des pages web à l'apparence soignée (couleur emplacement des éléments...etc)

JavaScript est un langage de programmation qui vous permet de mettre en œuvre des éléments complexes sur des pages Web.
Chaque fois qu'une page affiche des mises à jour de contenu en temps réel, des cartes interactives, des animations graphiques 2D / 3D ou un juke-box vidéo défilant, etc...
